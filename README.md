##Python Shopping Cart (CLI)

A command-line shopping cart application written in Python.
Users can add items, remove items, view their cart, and calculate totals with tax.

## Features
- Add items with price and quantity
- Remove items from cart
- View all cart items
- Automatic subtotal and tax calculation
- Menu-based CLI interface
- Input validation
